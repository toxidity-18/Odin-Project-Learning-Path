# Introduction 
- Listing using li tags 

## Unordered list 
- Using ul and li tags.
- Order does not matter at all . 
```html 
    <ul>
         <li>tea bags</li>
         <li>cooking flour</li>
    </ul>
  ```
  ## Ordered list
  - Using ol and li tags 
  - Order does matter a lot .
  ```html 
     <ol>
          <li>tea bags</li>
          <li>cooking flour</li>
      </ol>
  ```
